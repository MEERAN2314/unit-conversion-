About Unit Converter Pro

A comprehensive, professional-grade unit conversion library with beautiful web interface

Project Overview

The Unit Conversion Library is a Python-based tool designed to convert between different units of measurement with high precision and accuracy. This web interface provides an elegant, user-friendly frontend for the powerful conversion engine.

Built with modern web technologies including FastAPI and Jinja2 templates, it offers both a beautiful web interface and a comprehensive RESTful API for programmatic access.

API Documentation

Convert Units
POST /api/convert

Request body:

{
  "value": 205,
  "from_unit": "mm",
  "to_units": ["m", "cm", "in", "ft"]
}
Get Available Units
GET /api/units

Returns all available units organized by category.
